# Ahava Healthcare - Patient App

Patient-facing application for booking and tracking home healthcare visits.

## Features

- ✅ Patient registration and login
- ✅ Book home visits
- ✅ Track visit status
- ✅ View visit history
- ✅ Payment processing (Paystack)
- ✅ Real-time updates

## Getting Started

```bash
npm install
npm run dev
# Open http://localhost:3002
```
